Time sensitive observables are unique, but can be very useful when you want to capture data every so often. For example, if you only need to know about certain events or action every 10 seconds, you would normally use an interval. Intervals and observables don't exactly play well together, as you're combining two asynchronous functions together, which can yield some strange results. Luckily, RxJS has taken this into account with some fun functions: auditTime and bufferTime.

What if we don't just want to know the latest value of our subject though? What if we want to know all the values that were sent to our subject while we were waiting for our audit to happen? That is where bufferTime comes into play. bufferTime behaves similarly to auditTime, except that it will emit an array of all values sent to our subject while we were waiting for our time interval to pass.
